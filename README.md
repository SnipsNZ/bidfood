# United Food Co – Asian Kitchen Presentation

A 16:9 product presentation for United Food Co's Asian Kitchen noodle range, optimised for display on laptop or TV.

## Structure

```
ufc-site/
├── index.html
├── assets/
│   ├── style.css
│   ├── main.js
│   ├── fonts/
│   │   └── Gotham_Bold.otf
│   └── images/
│       ├── logo_white.png
│       ├── logo_black.png
│       ├── tonkotsubowl.png
│       ├── tonkotsubox.png
│       ├── wontonbowl.png
│       ├── wontonbox.png
│       ├── dandanbowl.png
│       └── dandanbox.png
└── README.md
```

## Deploying to GitHub Pages

1. Create a new GitHub repository (e.g. `ufc-presentation`)
2. Upload all files maintaining the folder structure above
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch` → `main` → `/ (root)`
5. Save — your site will be live at `https://yourusername.github.io/ufc-presentation/`

## Navigation

- **Arrow keys** — cycle through slides
- **Enter / Space** — open first product from overview
- **Escape** — return to overview
- **Nav bar tabs** — click to jump between products
- **Dots** — click to cycle overview slideshow

## Display

Best viewed full-screen (F11) at 16:9 resolution (1920×1080 or 1280×720).
